Generates a string based on a given Velocity template.

Fields:
  • file: Path to the template file.
  • encoding: The encoding to use when reading the template file.
  • text: A string containing the template to be used. This can be used instead of a file template.
  • parameters: Parameters needed to be passed to the Velocity engine when resolving the template.
Example:
node {
  def resolvedString = resolveTemplate(
    file: '/path/to/the/template.vm',
    parameters: [ some: 'thing'])
  echo resolvedString
}

node {
  def resolvedString = resolveTemplate(
    text: 'A template string for some${some}',
    parameters: [ some: 'thing'])
  echo resolvedString
}
