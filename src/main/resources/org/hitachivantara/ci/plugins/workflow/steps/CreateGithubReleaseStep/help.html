Retrieves the changelog between branches, commits, tags and SHAs

Fields:
  • credentialsId: ID by which the credentials are identified.
  • repository: Repository to connect to. Should be in the 'owner/repoName' format (example: 'pentaho/pentaho-kettle')
  • name: The name of the release.
  • text: The body text for the release.
  • update: Whether a release should be recreated if another release with the same name already exists. Defaults to 'true'
Example:
node {
  Boolean saved =
      createGithubRelease(
        credentials: 'jenkinsCredentialsId',
        repository: 'pentaho/pentaho-kettle'
        name: 'release-name',
        text: 'release-text',
        update: true
      )
}
