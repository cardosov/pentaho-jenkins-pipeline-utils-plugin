Process a Maven Project file. The returned object is a MavenModule.

Fields:
  • file: File to read.
  • activeProfiles: Optional: List of profile ids.
  • userProperties: Optional: Properties object.
Example:
node {
  MavenModule x = buildMavenModule file: 'pom.xml'
  assert x.id == 'com.pentaho.example:multimodule-parent'
}
