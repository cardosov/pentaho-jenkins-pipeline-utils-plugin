Deletes files or folders from the workspace.

Fields:
  • path: Path to the file or folder, relative or absolute.
  • regex: Regular expression to match for files or directories to delete starting in the current folder.
  • async: Perform deletion asynchronously. Only available for paths. Defaults to false.
Example:
node {
  delete(path: 'path/to/dir', async: true)

  dir('path/to/search') {
    delete(regex: '.*txt')
  }

  // if no path or regex are provided the current directory is considered
  dir('dir/to/delete'){
    delete(async: true)
  }
}
